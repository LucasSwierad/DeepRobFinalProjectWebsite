Motion-Aware Regularization for 4D Gaussian Splatting

Lightweight unsupervised static–dynamic decomposition via a learned motion-mask gate.

Author One,  Author Two,  Author Three

Abstract

This project studies static–dynamic separation in a modified 4D Gaussian Splatting (4DGS) codebase. The baseline 4DGS model represents a dynamic scene by storing Gaussian primitives in a canonical space and predicting time-dependent deformation before rasterization. However, the baseline does not explicitly indicate which Gaussians should remain static and which should participate in motion.

To address this, we add a lightweight motion-mask branch inside the deformation network. The mask is a learned scalar gate that modulates deformation strength and is optimized jointly with image reconstruction — with no ground-truth static/dynamic labels or external segmentation during training. We combine the original reconstruction objective with two additional regularizers: a binarization loss and a static-deformation loss.

Experiments on D-NeRF and HyperNeRF scenes show that the proposed method produces a more interpretable motion decomposition and can modestly improve reconstruction on strongly dynamic synthetic scenes. Results also reveal a consistent tradeoff between reconstruction quality and mask separability, and do not indicate universal gains on complex real-world data.

Method

We add a new scalar branch to the existing 4DGS deformation network. For each Gaussian i at time t, the motion mask is predicted from the same hidden feature used by the deformation heads:

Motion-Mask Forward Path
1Start from canonical Gaussian attributes: position, scale, rotation, opacity, SH color.
2Query the HexPlane field using canonical position and timestamp → spatiotemporal feature F_i(t).
3Project HexPlane feature through shared MLP → hidden feature h_i(t).
4Predict deformation heads (Δμ, Δs, Δr) and motion mask m_i(t) = σ(g_φ(h_i(t))).
5Apply gated deformation: μ_i(t) = μ_i⁰ + m_i(t) · Δμ_i(t)
6Rasterize deformed Gaussians and compute image reconstruction loss.

Training Objective

Two motion-specific regularizers are added on top of the baseline image reconstruction loss:

Binarization Loss

𝓛_bin = (1/N) Σ m_i(t)(1 − m_i(t))

Discourages persistent mid-range mask values, pushing the mask toward 0 or 1.

Static-Deformation Loss

𝓛_static = (1/N) Σ (1 − m_i(t)) ‖Δμ_i(t)‖₂

Penalizes motion in regions where the mask is small (predicted static).

Key Findings

  • +0.22 to +0.28 dB PSNR on Bouncingballs — consistent reconstruction gain on this strongly dynamic synthetic scene across all non-collapsed settings.
  • +0.22 dB PSNR on Jumpingjacks — moderate gain; heavier regularization (2e-3 / 2e-3) can slightly hurt image quality.
  • No PSNR gain on HyperNeRF Chickchicken — but stronger regularization produces a dramatically cleaner motion decomposition, showing that reconstruction quality and mask interpretability are separable objectives.
  • Collapse failure mode — excessive static-deformation weight (1e-2 / 1e-3) drives the model to a trivial all-dynamic solution (dynamic fraction = 1.0).

Related Links

Key related and concurrent works:

  • 4D Gaussian Splatting (baseline) — The baseline 4DGS framework this work builds on; uses a HexPlane deformation field over canonical Gaussians.
  • SDD-4DGS — A static-dynamic decoupling framework reporting +0.68 dB PSNR on D-NeRF; uses a persistent per-Gaussian dynamic coefficient rather than a predicted gate.
  • D-NeRF — Dynamic NeRF benchmark used for Bouncingballs and Jumpingjacks evaluation.
  • HyperNeRF — Real-world monocular dynamic scene dataset; Chickchicken scene used in this work.
